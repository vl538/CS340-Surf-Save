# CS340 Portfolio Project – Surf & Save

Group 40  
Vincent Le & Mykola Lopushenko

## Project Description
Surf & Save is a database-driven web application designed to manage an online retail system for surf-related products. The system allows users to manage customers, products, orders, and order items using full CRUD functionality.

## Technologies Used
- Node.js
- Express
- MySQL
- JavaScript
- HTML/CSS
